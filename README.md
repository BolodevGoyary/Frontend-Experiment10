# Frontend-Experiment10: API Data Fetching & Display (JavaScript Fetch API)

## Description
This experiment demonstrates how to fetch data from a public API using the JavaScript **Fetch API** and display it dynamically on a web page. The project uses **JSONPlaceholder** as the data source and displays user information including name, username, email, city, and company.

## Features
- Fetches data asynchronously from a public API
- Parses JSON responses
- Dynamically creates HTML elements to display data
- Responsive design using CSS Grid

## Project Structure
- index.html
- style.css
- script.js
- README.md

## How to Run
1. Clone or download the repository.
2. Open `index.html` in any web browser.
3. Click the **"Fetch Users"** button to fetch and display user data.

## Dependencies
No external dependencies. All code is written in HTML, CSS, and JavaScript.

## Notes
- The project displays the first 5 users fetched from the JSONPlaceholder API.
- You can modify the API endpoint or display more data as needed.
